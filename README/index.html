Here is the source code of the website internet2000.net, developed by INTERNET 2000 French agency

Tools we use

  • 11ty static site generator
  • Silex website builder
  • forestry headless CMS
  • snipcart e-commerce
  • github pages and github actions
  • Integromat to create a blog post for each instagram post on internet2000 on instagram

local installation

$ nvm i
$ npm i
$ npm run build

build and deploy

Useful env vars on the build server

  • URL optional website URL
  • BASE_URL optional base url
  • DEPLOY_TOKEN for deployment to branch gh-pages
